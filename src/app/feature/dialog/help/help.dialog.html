Allgemein

Mit BubbleVision können Sie visuell und explorativ Einsicht in deutsche Twitter-Gemeinschaften gewinnen. Auf diese Weise soll interaktiv ein Verständnis für das Konzept von Filterblasen vermittelt werden. Filterblasen sind ein Nebeneffekt personalisierter Netzwerke: Benutzern werden nur Inhalte gezeigt, die auch ihren Interessen entsprechen. Daraus resultiert häufig eine passive Polarisierung des eigenen Meinungsbildes, wenn man sich nicht bewusst mit einer diversen Menge an Meinungen ausseinandersetzt. Mehr erfahren Sie hier.

Bedienung

Sie können den Graphen bewegen, indem ihn Sie mit gedrückter Maustaste umherziehen. Um herein oder heraus zu zoomen, können Sie das Mausrad (auf Laptops die Scrollgeste) verwenden. Mit einem Klick auf ein Profil oder eine Gemeinschaft können Sie sich Zusatzinformationen anzeigen lassen.

Performanz

Die Darstellung des Graphens hat einen hohen Berechnungsaufwand und kann je nach Systemkonfiguration ein wenig ruckeln. Um die bestmögliche Leistung aus Ihrem Gerät zu holen, können Sie folgende Schritte ausprobieren:

  1. Google Chrome verwenden
  2. Die Einstellungen dieser Anwendung anpassen
  3. Den Browser aktualisieren
  4. Eine Grafikkarte einbauen
