Herzlich willkommen!

Allgemein

Mit BubbleVision können Sie visuell und explorativ Einsicht in deutsche Twitter-Gemeinschaften gewinnen. Gemeinschaften werden farblich hervorgehoben. Individuen sind schwarz markiert.

Bedienung

Sie können den Graphen bewegen, indem ihn Sie mit gedrückter Maustaste umherziehen. Um herein oder heraus zu zoomen, können Sie das Mausrad (auf Laptops die Scrollgeste) verwenden. Mit einem Klick auf ein Profil oder eine Gemeinschaft können Sie sich Zusatzinformationen anzeigen lassen.

Mehr

Zusätzliche Informationen können Sie im Hilfemenü erhalten. Viel Spaß bei der Erkundung!
