Herzlich willkommen!

Mit BubbleVision können Sie visuell und explorativ Einsicht in deutsche Twitter-Gemeinschaften gewinnen. Twitter ist ein soziales Netzwerk, mit dem Benutzer telegrammartige Kurznachrichten teilen können.

Profile mit ähnlichen Followern befinden sich in der Nähe voneinander. Gemeinschaften sind farblich hervorgehoben, Individuen schwarz. Profilbilder werden erst geladen, wenn Sie nah genug herangezoomt haben.

Sie können den Graphen bewegen, indem ihn Sie mit gedrückter Maustaste umherziehen. Um herein oder heraus zu zoomen, können Sie das Mausrad (auf Laptops die Scrollgeste) verwenden. Mit einem Klick auf ein Profil oder eine Gemeinschaft können Sie sich Zusatzinformationen anzeigen lassen.

Zusätzliche Informationen können Sie im Hilfemenü erhalten. Viel Spaß bei der Erkundung!
